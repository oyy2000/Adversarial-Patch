# adversarial-patch
PyTorch implementation of adversarial patch 
python==3.8.18 pytorch==2.0.0

Run attack:

- `python make_patch.py --cuda --netClassifier resnet18 --max_count 100 --image_size 32 --patch_type square --outf log`
